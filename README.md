# My Self App
This project was created for the UTS Mobile Computing Application Course
--

This project uses:
- MVVM Architecture
- View Binding
- Kotlin Coroutines
- Room Database
- View Model
- Live Data
- Mapbox SDK
- Repository
- ViewPager 2
- DiffUtil

The design for this project can be accessed at https://www.figma.com/file/an62oADf0X0IgCxapsqY0o/My-Self-Apps?node-id=1%3A3
